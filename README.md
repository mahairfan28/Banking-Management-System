# 🏦 Banking Management System
A complete banking management system built in Python for my Programming for Business final project. Features account creation, deposits, withdrawals, fund transfers, printable receipts, financial reports, and data visualization using matplotlib.

## Project Information

- **Semester:** Second Semester Final Project

## Features
- Create bank accounts
- Deposit money
- Withdraw money
- Transfer money
- Print receipts
- View reports
- See charts (matplotlib)

## How to Run

1. Install Python from python.org
2. Open Command Prompt / Terminal
3. Type: `pip install matplotlib`
4. Type: `python banking_system.py`
